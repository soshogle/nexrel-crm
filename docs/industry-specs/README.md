This folder is the source of truth for industry-specific AI employees, workflows, tools, and UI behavior.

Scope for Phase 1:

- Lock non-negotiable guardrails before implementation.
- Define the spec matrix for all industries in scope.
- Preserve all existing customer data and user experience.

Out of scope for Phase 1:

- Database schema changes.
- Visual redesign.
- Destructive migrations.
